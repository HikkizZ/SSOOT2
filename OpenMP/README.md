## OpenMP

### Instalacion de la libreria

sudo apt-get install libomp-dev

### Compilacion

Se debe utilizar el comando make estando en la carpeta del OpenMP

### Ejecucion

El siguiente comando debe ser utilizado para ejecutar el programa:

./main ../imagenes/tyler.jpg ../imagenes/output_openmp.jpg

#### Valores a entregar
programa - imagen de entrada - imagen de salida
